Tweeter is a work-in-progress Twitter-like social application built upon Clean 
Architecture and Clean Code principles.

Checkout the live demo: https://tweeter-dfa01.web.app/.

## Available features
- Post tweets
- Browse users
- Follow users

## Planned features
- Offline support
- Like
- Retweet
- Confirm email address
- Reset a password
- Mobile-friendly UI

## Development setup
```bash
# install deps
npm install

# start a development server at localhost:3000
npm run dev

# run a production build
npm run prod

# run all tests
npm test

# run unit tests
npm run test:unit

# start firebase emulators
npm run serve

# run database integration tests (requires emulators to run)
npm run test:db
```
